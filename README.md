## <p align="center"> RuPa: A Rust panel for Linux management </p>

### 痛点

- 以 Python 动态语言为主的宝塔面板，如今非常臃肿，因政策限制、广告等让人痛心疾首。
- 以 Go 为主的 1panel 几乎重构了宝塔面板，但严重依赖 docker，虽优秀但不够彻底。

### 新方案

- 基于 Rust 语言开发，仅保留高频使用功能。
- 零依赖、更小、更快、更安全。
